## Task  List ##

Aplicación creada con Node.js para crear y eliminar tareas utilizando el modulo "readline" para interactuar con el usuario



## Funcionalidades 

1. Crear Tareas
2. Eliminar Tareas
3. Completar Tareas
4. Ver Lista de Tareas



## Requisitos 

- Node.js instalado

1. Clonar este repositorio:
```
